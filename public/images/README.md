# Images folder

Drop your custom images here so the site can use them.

Expected files (used directly by the code):

- `logo.png`         — לוגו השרת (מופיע ב-Header וב-Footer). מומלץ ~256x256 px, רקע שקוף.
- `placeholder.png`  — תמונת ברירת מחדל לחבילה כשאין תמונה ב-Tebex.

Optional (for future use):
- `hero.png`         — תמונת רקע לעמוד הבית.
- `bg-grid.png`      — טקסטורה / רקע.

After dropping the files into this folder, refresh the browser (no rebuild needed in dev mode).
Reference them in code as absolute paths from root, e.g. `/images/logo.png`.
